# 🎯 AI 소비 습관 코치 (for 대학생)

> 대학생과 사회초년생의 건강한 금융 습관 형성을 돕는 **AI 기반 소비 코칭 플랫폼**

---

## 📘 Overview

기존 가계부 앱이 ‘기록’에 머무는 한계에서 벗어나,  
AI가 사용자의 소비 데이터를 분석하고 **개인 맞춤 코칭**을 제공하는  
새로운 금융 트레이닝 서비스를 목표로 합니다.

**핵심 목표**
- 월평균 불필요 지출 15% 절감
- 6개월 내 입력 지속률 50% 이상
- 1개월 리텐션 20% 이상

---

## 🔍 Problem Definition

| 문제 | 설명 |
|------|------|
| **20대의 45.8%** | 체계적 지출 관리 어려움 (금융위, 2022) |
| **기존 앱 이탈률 70%** | 3개월 내 기록 중단 (앱애니 조사) |
| **금융 문해력 공백** | “얼마 썼다”는 알지만 “어떻게 개선할지”는 모름 |

---

## 👥 Target User

**Primary Persona:**  
김지민 (22세, 대학생)  
- 월 수입: 70만 원 (용돈 + 아르바이트)  
- Pain Point: 카페·배달비 과소비, 가계부 2주 이상 지속 불가  
- Goal: 학기 말까지 30만 원 저축  

**특징**
- 자기계발 욕구 강한 Z세대  
- 금융 이해력 평균 이하 (65.5점)  
- 실생활 속 *쉽고 직관적인 소비 코칭* 니즈

---

## 💡 Core Concept

> “나의 소비를 진단하고 코칭해주는 AI 금융 트레이너”

**핵심 구조**
입력 → 자동 분석 → 시각화 → 개인화 코칭 → 실행 피드백


친구처럼 자연스럽게 대화하고 행동 변화를 유도하는  
*AI 코치 경험* 중심 UX 설계

---

## ⚙️ Key Features

| 구분 | 기능명 | 설명 | 단계 | 비고 |
|------|---------|------|------|------|
| **데이터 기반 구축** |||||
| ① 간편 지출 입력 | FAB 버튼으로 2탭 이내 입력 (금액·내용·카테고리 최소화) | MVP | UX 핵심 |
| ② 데이터 가져오기 | CSV/Excel 업로드 + 필드 매핑 | MVP | 추후 오픈뱅킹 연동 |
| ③ AI 자동 분류 | 키워드 기반 → ML 모델로 고도화 | MVP | 피드백 학습 구조 |
| **핵심 경험 제공** |||||
| ④ 대시보드 | 월/주별 분석, 예산 대비, Top3 소비 | MVP | 단순·명확 시각화 |
| ⑤ 소비 내역 조회 | 최신순, 검색, 필터, 수정·삭제 | MVP | 데이터 신뢰성 확보 |
| **행동 변화 유도** |||||
| ⑥ 개인화 인사이트 | “지난달 대비 식비 +15%”, “카페 줄이면 3만 원 절약” | Phase 2 | 알림 톤 선택 (코치형/친구형) |
| ⑦ 절약 챌린지 | “배달 1번만” 목표 + 배지/포인트 보상 | Phase 2 | Retention 핵심 |
| **신뢰 & 보안** |||||
| ⑧ 데이터 보안 | 암호화/익명화, 개인정보 처리 절차 | MVP | 필수 보안 조치 |
